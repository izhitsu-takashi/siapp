# フォントファイルの配置

このフォルダに日本語フォントファイルを配置してください。

## 推奨フォント

- **Noto Sans JP Regular** (推奨)
  - ダウンロード: https://fonts.google.com/noto/specimen/Noto+Sans+JP
  - ファイル名: `NotoSansJP-Regular.ttf`
  - ライセンス: SIL Open Font License (商用利用可)

## フォントファイルの配置方法

1. 上記のリンクからNoto Sans JPをダウンロード
2. `NotoSansJP-Regular.ttf` をこのフォルダ（`src/assets/fonts/`）に配置
3. アプリケーションを再起動

## 注意事項

- フォントファイルが存在しない場合、PDFへの日本語入力はできません（英数字のみ）
- フォントファイルのサイズは約2-3MB程度です
- フォントファイルを配置した後、アプリケーションを再ビルドする必要があります
