# 修正内容：Geminiモデル名の変更

## 問題
エラーメッセージ：
```
models/gemini-pro is not found for API version v1beta, or is not supported for generateContent.
```

## 原因
`gemini-pro`モデルは非推奨となり、最新のGemini APIでは使用できなくなりました。

## 修正内容
モデル名を`gemini-pro`から`gemini-1.5-flash`に変更しました。

### 変更前
```typescript
const model = genAI.getGenerativeModel({model: "gemini-pro"});
```

### 変更後
```typescript
const model = genAI.getGenerativeModel({model: "gemini-1.5-flash"});
```

## デプロイ完了
✅ 修正をデプロイしました。

## 動作確認
1. アプリを開いて「ナレッジ」タブをクリック
2. チャット機能をテスト
3. メッセージを送信して、正常に応答が返ってくることを確認

## 参考
- `gemini-1.5-flash`: 高速で低コスト、一般的なチャット用途に適している
- `gemini-1.5-pro`: 高精度、複雑なタスクに適している

今回のチャット機能には`gemini-1.5-flash`を使用しています。
